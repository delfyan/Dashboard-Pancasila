Tentang Dashboard

Tujuan

Dashboard ini dirancang untuk memvisualisasikan dan menganalisis ketimpangan pendidikan di 38 kabupaten/kota Provinsi Jawa Timur menggunakan pendekatan statistik spasial (Moran's I, LISA) dan analisis komponen utama (PCA).

Sumber Data

Data indikator pendidikan bersumber dari Badan Pusat Statistik (BPS) Jawa Timur, mencakup Angka Partisipasi Sekolah (APS), Rata-rata Lama Sekolah (RLS), Angka Melek Huruf, dan Angka Partisipasi Kasar (APK).

Metodologi

Analisis mencakup (1) Konstruksi indeks pendidikan komposit berbasis PCA, (2) Uji Moran's I global untuk autokorelasi spasial, (3) Pemetaan LISA untuk identifikasi klaster lokal, dan (4) Interpretasi pola spasial ketimpangan.

Teknologi

Dibangun dengan HTML5, CSS3, dan JavaScript murni. Visualisasi menggunakan Chart.js dan peta interaktif dengan Leaflet.js. Dashboard dapat dijalankan langsung di GitHub Pages tanpa backend.
